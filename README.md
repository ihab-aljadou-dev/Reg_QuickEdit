# Reg_QuickEdit

Reg_QuickEdit is a personal Windows utility developed with Longtion Application Builder.

The project was created to simplify working with the Windows Registry through a graphical user interface and additional automation features.

## Features

* Browse Windows Registry keys and values
* Search for registry entries
* Add and delete keys
* Add and delete values
* Read registry data
* Open registry locations directly
* Integration of CMD commands and batch scripts

## Technologies

* Longtion Application Builder
* Windows CMD
* Batch Scripts (.bat)

## Project Status

Personal learning project developed to gain practical experience with Windows administration tools and application development.
